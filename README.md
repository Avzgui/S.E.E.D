## S.E.E.D ##

Seed for Environments and Entities Development, or S.E.E.D, is a game engine
developed in C++.

S.E.E.D is a combination of three engines :

- A 3D Engine.
- A Physics Engine, based on Newton Dynamics.
- An AI Engine, based on a Multi Agents System.
